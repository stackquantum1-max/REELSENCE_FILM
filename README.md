# Reelsence_Film Website
---
A modern, responsive portfolio website for professional photography and videography services.  
Showcases projects, services, client testimonials, and a contact form with a clean, interactive design.

## Features
- **Responsive Navbar** – Sticky navigation with mobile dropdown
- **Hero Section** – Eye-catching introduction
- **Portfolio Gallery** – Interactive image grid with modal preview
- **Services Section** – Highlights core offerings
- **Testimonials Slider** – Auto-sliding and navigable
- **Contact Form** – User-friendly for client inquiries
- **Footer** – Social media links and newsletter subscription

---
## Technologies Used
- HTML5 – Structure
- CSS3 – Styling & responsive design
- JavaScript – Interactivity & DOM manipulation
- Google Fonts – Montserrat & Poppins
- Font Awesome – Icons
